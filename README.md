# IdentitySecure

IdentitySecure é uma aplicação desktop em Java com interface gráfica (Swing) voltada para estudo de segurança e organização de sistemas.

## Funcionalidades

- Consulta e validação de CPF (simulada)
- Scanner simples de portas (21, 22, 80, 443)
- Informações do sistema (arquitetura, núcleos, memória JVM, SO, IP local)

## Tecnologias

- Java
- Swing
- IntelliJ IDEA
- Git e GitHub

## Estrutura

src  
├── ui  
├── service  
└── model  

## Objetivo

Projeto desenvolvido para prática de:
- Programação orientada a objetos
- Interface gráfica em Java
- Organização em camadas
- Conceitos básicos de segurança
